创建一个空集合必须用 set() 而不是 {}，因为 {} 是用来创建一个空字典。

字典是一种映射类型，字典用 { } 标识，它是一个无序的 键(key) : 值(value) 的集合。键(key)必须使用不可变类型。在同一个字典中，键(key)必须是唯一的。

元组（）里面的字符串要用单引号‘ ‘引用，print输出tuple元素时使用[]。

创建空集合使用set()，{}是创建空字典，-是差集，|是并集，&是交集，^是对称差集，对称差集是并集减去差集a=set(‘sbc’)print(a)，输出结果为{‘s’,’b’,’c’}。a=set({‘sbc’})，print(a)，输出结果为{‘sbc’}。
列表是有序的，字典是无序的。1、字典是一种映射类型，它的元素是键值对。2、字典的关键字必须为不可变类型，且不能重复。3、创建空字典使用 {}。

在字节字符串中，每个字符都表示一个单独的字节，可以通过索引访问。例如，byte_string[0] 将返回字节字符串中的第一个字节。

a, b, c = 5, 3, 2 是一种赋值语句的写法，相当于将 a = 5, b = 3, c = 2 三个赋值语句写在一起。除法赋值语句 a /= b 会将 a 的类型变为浮点型（float）

print('a =', a, end='')这一行代码打印了字符串 'a ='，然后使用逗号,连接了变量 a 的值，并且通过 end='' 参数指定了输出结束时不添加换行符。end='' 参数使得打印的内容不会换行，而是连续输出。

字符串是 Python 中最常用的数据类型。在 Python 中没有单字符类型，单个字符只是长度为 1 的字符串。切片时左闭右开

在 Python 中，从字符串中更新或删除字符是不被允许的，这些操作会导致产生错误，虽然我们可以使用内置的 del 关键字来删除整个字符串，但是字符串是一个不可改变的整体，因此字符串在分配地址后无法更改，之后能通过分配新的字符串给同一名称的变量实现改变。

命名规则-驼峰命名法

1.小驼峰命名：第一个单词以小写字母开头，其余单词首字母大写，如：theFirstName

2.大驼峰命名：所有单词首字母均大写，如：TheFirstName

3.一般我们定义一个标识符时，通常使用下划线_，如：card_add、card_pop

运算符 功能 描述

is not 判断是否不是同一个对象 a is not b，类似于 id(a) != id(b)，返回一个布尔值代表 a 和 b 是否不是同一个对象

拓展知识：is 与 == 的区别

is 用于判断两个引用是否为同一个对象，== 则判断两个引用的值是否相等，但是要注意的是，对 Python 的整数而言，小部分值相等的常量（整数区间 [-5, 256] 的范围内的常量）引用的都是同一个对象（类似 Java 中的常量池）。在Python中，小整数缓存的范围通常是-5到256，但实际的范围可以因Python解释器的不同版本和实现而有所差异。

Tips:

1、在实际编写比较长的计算式时，可以使用多个小括号来设置优先级，哪怕不知道确定的优先级，只要带着小括号，小括号内的表达式就会优先计算，也方便后期代码的维护。

2、逻辑与或和按位与或的优先级类似，都是与运算优先级大于或运算优先级。

数学函数

函数 返回值 ( 描述 )
abs(x) 返回数字的绝对值，如 abs(-10) 返回 10
ceil(x) 返回数字的上入整数，如 math.ceil(4.1) 返回 5
exp(x) 返回 e 的 x 次幂 (��e**x) 如 math.exp(1) 返回 2.718281828459045
fabs(x) 以浮点形式返回数字的绝对值，如 math.fabs(-10) 返回 10.0
floor(x) 返回数字的下舍整数，如 math.floor(4.9) 返回 4
log(x) 如 math.log(math.e) 返回 1.0，math.log(100,10) 返回 2.0
log10(x) 返回以 10 为基数的 x 的对数，如 math.log10(100) 返回 2.0
max(x1, x2,…) 返回给定参数的最大值，参数可以为序列。
min(x1, x2,…) 返回给定参数的最小值，参数可以为序列。
modf(x) 返回 x 的整数部分与小数部分，两部分的数值符号与 x 相同，整数部分以浮点型表示。math.modf(1.5) 返回 (0.5, 1.0)
pow(x, y) x ** y 运算后的值。math.pow(2, 7) 返回 128.0
round(x, n) 返回浮点数 x 的四舍五入值，如给出 n 值，则代表舍入到小数点后的位数。其实准确的说是保留值将保留到离上一位更近的一端。
sqrt(x) 返回数字x的平方根。math.sqrt(9) 返回 3.0
